This is a small web game using JavaScript. Player has given max 20 score at the begnning of the game. They have to guess the number between 1 to 20 and check the number wheather the guessed number is correct or not. For each correct guess number player's score will be increased by 1 and for each wrong guess score will be decreased by 1. Minimum score required for playing the game is 1, below this the Game is Over.
